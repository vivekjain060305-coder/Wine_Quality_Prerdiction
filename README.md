Wine Quality Prediction using Machine Learning

1. Project Title
Wine Quality Prediction using Machine Learning

2. Project Description
This project focuses on predicting the quality of wine using various Machine Learning algorithms. The prediction is based on physicochemical properties such as acidity, pH, alcohol content, sulphates, etc.
The project implements and compares multiple models including Linear Regression, Decision Tree, K-Nearest Neighbors (KNN), Support Vector Machine (SVM), Artificial Neural Networks (ANN), and Polynomial Regression.
The objective is to identify the most accurate and reliable model for predicting wine quality.

3. Objectives
•	To apply Machine Learning techniques for wine quality prediction
•	To compare different ML models
•	To improve prediction accuracy using preprocessing techniques
•	To analyze model performance using evaluation metrics

4. Software Requirements
•	Python (Version 3.8 or above)
•	Jupyter Notebook / Anaconda
•	Operating System: Windows / Linux / Mac

•	numpy

•	tensorflow (for ANN)
6. Dataset Information
•	Dataset Name: Wine Quality Dataset
•	Source: Kaggle / UCI Repository
•	File Used: wine-quality-white-and-red.csv
Input Features:
•	Fixed Acidity
•	Volatile Acidity
•	Citric Acid
•	Residual Sugar
•	pH
•	Sulphates
•	Alcohol
Output:
•	Wine Quality Score

8. Steps to Run the Project
Step 1: Install Python
Download and install Python from the official website.

Step 2: Install Required Libraries

Step 3: Open Project Folder
Extract the project folder and open it.

Step 4: Launch Jupyter Notebook
Run the following command:

Step 5: Open Any Model File

•	KNN
•	SVM

Step 6: Run the Code
Click on Run All Cells in the notebook.

Step 7: View Results

9. Output
The system predicts the quality of wine based on input features and compares performance of different models.

10. Results Summary
•	Decision Tree gives best overall performance
•	KNN provides consistent results
•	ANN and SVM perform well for complex patterns
•	Linear Regression performs poorly for classification

11. Conclusion
Machine Learning models can effectively predict wine quality. Among all models, Decision Tree performed best in terms of accuracy and stability.
